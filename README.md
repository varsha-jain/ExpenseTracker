# ExpenseTracker
Why use plain old excel sheet to manage your expenses!? Instead use simple android application Expense Tracker to manage your daily expenses easily and effectively. This app allows you to create your account so that all your expenses are kept to yourself even if someone else uses your phone. It allows you  to add/view expenses and where you happened to spend more. So enjoy this app!
